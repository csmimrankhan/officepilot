## Submission Instructions

### Option A: In-App Bug Report (Recommended)

1. Go to **Settings → Bug Report**
2. Fill in the form
3. Check "Include logs" and/or "Include screenshot" as appropriate
4. Submit

### Option B: Email to Pilot Coordinator

Send the completed template to your pilot coordinator.

### Option C: Manual Diagnostic Package

1. Go to **Settings → Bug Report**
2. Click **Download Diagnostic Package** without submitting
3. Attach the ZIP file to your email or ticket

---

*Note: All bug report data stays local. If you choose to share diagnostic packages, passwords, tokens, API keys, and email addresses are automatically redacted. No data is sent to external servers without your explicit consent.*
