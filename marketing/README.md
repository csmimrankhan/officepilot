## Capture Notes

- Use browser DevTools at 1280x800 viewport.
- Ensure the app is in demo mode (`DEMO_MODE=true`) with sample data.
- Do not include real invoice data in screenshots.
- Save as PNG in this directory with the same base name (e.g. `screenshot_invoice_review.png`).
- Update the references in `frontend/public/landing.html` after capture.
