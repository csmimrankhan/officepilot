# OfficePilot AI

**Universal Voice Accountant Agent — Tell it what to do, it plans, shows steps, asks approval, executes safely, and remembers workflows.**

OfficePilot AI is a Windows-first desktop application that automates accounting work. You tell it what to do by voice or text, it plans the task, shows steps for your approval, executes safely, and remembers workflows for later. Works with Excel, Gmail, QuickBooks, Xero, and any web-based accounting platform — all while keeping your data local-first.

---

## Features

- **Voice & Text Commands** — Speak or type in English, Roman Urdu, or multiple languages. "Invoice download karo", "create Excel summary", "email sa invoice download karo" — the agent understands.
- **Task Planning** — Every command becomes a step-by-step plan. See exactly what the agent will do before it does it.
- **Approval Gates** — Every action requires your approval. Dry-run mode previews changes without touching your files.
- **Gmail Read-Only Automation** — Search emails, preview invoices, download attachments. Send/delete/modify is permanently blocked by a safety gate.
- **Real Excel Automation** — Auto-detect accounting columns, create summary sheets, generate Daily Invoices workbooks. Your original files are never modified (backup-first).
- **QuickBooks & Xero Sync** — Read-only sync of accounts, customers, invoices. Review before importing.
- **Workflow Memory** — Save workflows and replay them. "Repeat yesterday's workflow" works in English or Roman Urdu.
- **Browser Automation** — Navigate accounting platforms, export reports, download data. Sensitive fields (passwords, tokens) are auto-redacted.
- **Workflow Recorder** — Record your actions, convert to reusable skills, approve and save.
- **Desktop Control** — Read screen context, open folders/files, click and type with approval. Banking/password-manager apps are blocked by default.
- **Enterprise Safety** — Role-based permissions (5 roles, 18 permissions), audit logs, kill switch, version history with restore, file snapshots.
- **Local-First** — All data stays on your machine. No cloud dependency. Full privacy dashboard.
- **Demo Mode** — Built-in demo data and a 15-step guided onboarding for evaluation.
- **Auto-Update** — Tauri-based updater for seamless beta releases.

## Quick Start

### Prerequisites

- **Python 3.11+**
- **Node.js 20+** with npm
- **Rust toolchain** (only for building the Tauri desktop bundle; not needed for dev)

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

The API runs at `http://127.0.0.1:8000`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The UI opens at `http://localhost:5173`.

### Tests

```bash
cd backend && python -m pytest -q
cd frontend && npm test -- --run
```

### Desktop Bundle (requires Rust)

```bash
cd desktop/tauri
cargo tauri build --bundles msi,nsis
```
